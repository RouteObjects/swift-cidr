# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

- Initial public release of the `CIDR`, `CIDRConfig`, and `CIDRPOSIX`
  modules.
- Family-bound IP address, network, and prefix-length currency types.
- Mixed-family wrappers for boundary APIs.
- Transport endpoint modeling with `TransportPort` and `IPEndpoint`.
- Parser and currency benchmarks with threshold files for regression checks.
