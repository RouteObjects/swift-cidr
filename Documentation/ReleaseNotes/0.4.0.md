# swift-cidr 0.4.0

Status: Draft release notes; tag only after review and merge.

## Summary

`swift-cidr` 0.4.0 adds `AutonomousSystemNumber`, a canonical value-semantic
representation of the complete four-octet AS number space.

## Highlights

- Adds `AutonomousSystemNumber`, backed by `AF.ASN.Storage`.
- Parses and formats the RFC 5396 bare `asplain` decimal representation.
- Provides numeric `Codable`, `Hashable`, `Comparable`, and `Sendable`
  behavior.
- Preserves `AF.ASN` and the `ASN` alias as address-family marker types.
- Keeps RPSL `AS`-prefixed text, registry status, and routing-policy validation
  outside the numeric core value.

## Compatibility

This is an additive public API release. It does not change the existing IP,
prefix, network, endpoint, multicast, POSIX, or SwiftNIO APIs. No platform or
Swift tools version increase is required.

## Validation Before Tagging

- Run `./scripts/test.sh`.
- Run `./scripts/benchmarks.sh build` and `./scripts/benchmarks.sh check`.
- Run `./scripts/linux-test.sh` and `./scripts/linux-test.sh benchmark-build`.
- Confirm the `CIDR`, `CIDRPOSIX`, and `CIDRNIO` products build.
- Create the SSH-signed annotated `0.4.0` tag with message
  `swift-cidr 0.4.0` only after the feature branch has been reviewed and merged.
